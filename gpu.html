Tarjetas Gráficas (GPU)

Qué es una GPU

La GPU (Graphics Processing Unit) está diseñada para procesar en paralelo operaciones gráficas y ahora para cargas de ML. Las GPUs modernas cuentan con núcleos especializados (shader cores, tensor cores).

Tipos

  • Gaming: GeForce (NVIDIA), Radeon (AMD).
  • Data Center / IA: NVIDIA A100 / H100, GPUs con soporte para precisión mixta y mucha VRAM.

Marcas

NVIDIA domina IA con tensor cores; AMD compite fuerte en rasterizado y relación precio/eficiencia.

Comparaciones prácticas

NVIDIA GeForce RTX 4090

Tope de rendimiento gaming; también es potente en inferencia si se usa correctamente. Elevado consumo y precio.

NVIDIA A100 / H100

Diseñadas para centros de datos: memoria masiva, NVLink, optimizadas para entrenamiento.

AMD Radeon RX 7900

Buena competencia en rasterizado, mejor relación precio/rendimiento en algunos escenarios.

Consejos

Para gaming: busca GPUs con alta frecuencia y suficiente VRAM para texturas. Para IA: prioriza VRAM y soporte de frameworks (CUDA vs ROCm).
